## GitHub ID @kxxt

### 本月工作总结

kxxt 在 2026 年 5 月继续推进 Chromium 的 RISC-V 支持, 持续维护 Chromium RISC-V CI/CD 系统, 向 Chromium 和 Swiftshader 项目提交 CL 共 4 个,
为 Chromium 开启了 highway 的 RVV runtime dispatch, 修复了 Swiftshader 在 RISC-V 上的编译问题.
除此之外, kxxt 还向内核 BPF 列表反馈了一个影响 RISC-V 的 BPF local storage bug.
