### 本月工作总结

本月工作主要围绕 `sipeed/picoclaw` 与 `sipeed/picoclaw_docs` 展开，重点包括 Pico 协议演进、Provider 能力扩展、历史记录一致性修复、模型管理体验完善，以及文档补充与社区问题排查。

其中较为重要且难度较高的工作，主要集中在三类：一是持续推进对话协议与多 Provider 兼容性演进，包括分离 thought/reasoning 消息、引入原生 Gemini Provider、补齐结构化 `tool_calls` 的跨端传递与展示，并统一相关消息 kind 表示；二是修复历史记录与推理内容持久化问题，解决 Web Chat 刷新后消息不一致、DeepSeek thinking mode 后续轮次异常，以及 SeaHorse SQLite 丢失 `reasoning_content` 等问题；三是完善基础能力与可用性，包括 launcher / gateway host 绑定重构、网络错误分类与 fallback 增强、OAuth `--no-browser` 登录支持，以及基于显式 provider 元数据的模型管理改造。这些工作覆盖 CLI、后端与 Web UI 多个层面，对稳定性和可配置性提升较为明显。

除代码提交外，本月还持续参与 GitHub Issue 回复、问题复核与使用指引澄清，并完成了一份社区运营相关的技术误导复核报告。
